# 🐷 도방육종 텔레그램 봇 — 설치 가이드

## 필요한 것
- PC 또는 라즈베리파이 (24시간 켜둘 수 있는 것)
- Python 3.10 이상
- 텔레그램 봇 토큰 (BotFather에서 재발급)
- 노션 통합 토큰

---

## 설치 순서

### 1. Python 패키지 설치
```bash
pip install -r requirements.txt
```

### 2. 환경변수 파일 설정
```bash
cp .env.example .env
```
`.env` 파일 열어서 토큰 직접 입력:
```
TELEGRAM_BOT_TOKEN=재발급받은_새_토큰
NOTION_TOKEN=노션_통합_토큰
```

### 3. 노션 통합 토큰 발급 방법
```
notion.so → 설정 → 연결 → 통합 개발 → 새 통합 만들기
→ 이름: 도비봇 → 제출
→ 내부 통합 토큰 복사
```
그리고 각 DB 페이지에서 → 우측 상단 ··· → 연결 추가 → 도비봇 선택

### 4. 봇 실행
```bash
python bot.py
```

### 5. 텔레그램 그룹 설정
```
1. 텔레그램에서 새 그룹 만들기
2. 그룹에 봇 추가: @dobang_farm_bot (본인 봇 아이디)
3. 봇을 관리자로 설정
4. 썬, 콰 초대
5. /start 입력하면 버튼 메뉴 표시됨
```
